# Regularization-Based Defense for LTE/4G

This repository contains the implementation of the **regularization-based traffic defense** proposed in the following paper:

> Jinjin WANG, Zishuai Cheng, Mihai Ordean, Baojiang Cui. "What-App? App Usage Detection Using Encrypted LTE/5G Traffic" In 26th Privacy Enhancing Technologies Symposium (PETS 2026).

# About
The proposed defense is designed to protect mobile users from cellular traffic fingerprinting attacks, where an adversary infers application usage or in-app activities by analyzing encrypted LTE/5G traffic metadata.

Our approach introduces a traffic-shaping layer at the PDCP level, applicable to both the UE and the base station. By transmitting fixed-size packets at fixed intervals, supplemented with padding or dummy packets when necessary, the defense hides distinctive traffic patterns such as burstiness, packet size variations, and timing correlations.
